# Roadmap

This document tracks large features on the Hexagon engineering roadmap. This is not an authoritative project management document but serves to reduce questions and provide visibility.

It should not be assumed that the architecture is 'set' based on this roadmap. We have many other features that might appear.

3. Note any dependencies
4. Suggest target timeline

Maintainers review proposals quarterly and update this document.
