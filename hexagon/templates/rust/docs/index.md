# Hexacore

Workspace template and tooling for Phenotype kits.

## Overview

Hexacore is a curated collection of reusable Rust libraries for modern software development, organized as a Rust workspace.

## Kits

This workspace includes the following kits:

| Kit | Description |
|-----|-------------|
| clikit | CLI utilities |
| agentkit | Agent abstractions |
| evalkit | Evaluation helpers |
| taskkit | Task management |
| configkit | Configuration management |
| authkit | Authentication |
| cachekit | Caching |
| logkit | Logging |
| tracingkit | Tracing |
| metrickit | Metrics |
| eventkit | Event handling |

## Architecture Principles

- **SOLID** - Single Responsibility, Dependency Inversion
- **DRY** - Shared abstractions
- **PoLA** - Descriptive error types
