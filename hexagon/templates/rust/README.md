## Stack

- **HTTP**: Axum 0.7 (hyper-based, ergonomic middleware)
- **Database**: SQLx 0.8 (async, compile-time checked SQL)
- **Runtime**: Tokio (async runtime)
- **Errors**: thiserror (custom error types)
- **Observability**: tracing (structured logging)
- **Testing**: tokio-test, sqlx-test

## Usage

```bash
cargo new my-project --template hexagon/templates/rust
cd my-project
cargo run
```

## Project Structure

```

в”ңв”Җв”Җ domain/           # Core business logic
в”Ӯ   в”ңв”Җв”Җ entities/     # Domain entities
в”Ӯ   в”ңв”Җв”Җ ports/        # Domain ports (traits)
в”Ӯ   в””в”Җв”Җ services/     # Domain services
в”ңв”Җв”Җ application/      # Use cases
в”Ӯ   в”ңв”Җв”Җ ports/        # Application ports
в”Ӯ   в””в”Җв”Җ use_cases/    # Business operations
в””в”Җв”Җ infrastructure/   # External concerns
    в”ңв”Җв”Җ adapters/     # Adapter implementations
    в”Ӯ   в”ңв”Җв”Җ persistence/  # Database adapters
    в”Ӯ   в””в”Җв”Җ web/          # HTTP handlers
    в””в”Җв”Җ config/       # Configuration
```
