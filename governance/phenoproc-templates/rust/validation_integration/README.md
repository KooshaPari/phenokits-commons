# Validation Integration Template

This template shows how to integrate `phenotype-validation` into your Rust project.

## Files

- `Cargo.toml` - Dependencies
- `src/config.rs` - Configuration with validation
- `tests/validation_tests.rs` - Validation tests

## Quick Start

1. Copy these files to your project
2. Define your configuration structs
3. Create validators using the fluent API
4. Validate at load time and runtime
