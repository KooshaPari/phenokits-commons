# Python libs — absorbed into phenotype-python-sdk

Python kit libraries formerly under `PhenoKits/libs/python/` live in
[`KooshaPari/phenotype-python-sdk`](https://github.com/KooshaPari/phenotype-python-sdk)
under `packages/`.

| Former PhenoKits path | Canonical package |
|-----------------------|-------------------|
| `libs/python/pheno-cli-builder/` | `packages/pheno-cli-builder/` |
| `libs/python/pheno-cli-kit/` | `packages/pheno-cli-kit/` |
| `libs/python/phenotype-logging/` | `packages/phenotype-logging/` |
| `libs/python/phenotype-py-kit/` | `packages/phenotype-py-kit/` |
| `libs/python/phenotype-testing/` | `packages/phenotype-testing/` |
| `libs/python/phenotype-id/` | `libs/python/phenotype-id/` (in-repo; polyglot) |

`phenotype-config` app config: `configs/app/phenotype-config/` (ported from PhenoKits)
or `packages/phenotype-config/` in the Python SDK monorepo.
