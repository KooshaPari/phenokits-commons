# PhenoProc governance templates port

**Source:** `PhenoProc/crates/phenotype-governance/{templates,configs}`  
**Date:** 2026-06-17

Templates and universal CI/governance configs from PhenoProc absorbed for
per-repo tailoring via phenokits-commons (scaffold boundary).

## Paths

- `governance/phenoproc-templates/` — Rust/TS/Python CI templates, BDD integration scaffolds
- `governance/phenoproc-configs/` — language-specific governance configs

## Consumer

New repos should copy from here or from HexaKit `.template.*` — not from archived PhenoProc.
